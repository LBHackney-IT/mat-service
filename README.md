## Getting Started

First install dependencies:

```bash
npm install
```

To run the next.js development server:

```bash
npm run dev
```

To run serverless offline:

```bash
npm run build
sls offline
```

To run unit tests:

```bash
npm run test:unit
npm run test:unit:watch
npm run test:unit:coverage
```

To run integration tests:

```bash
npm run build
sls offline
npm run test:integration
```
